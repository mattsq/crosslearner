crosslearner.cross_validate_acx(loader: DataLoader, mu0: Tensor, mu1: Tensor, *, model_config: ModelConfig, training_config: TrainingConfig, folds: int = 5, device: str | None = None, log_dir: str | None = None, seed: int = 0) → float[source]¶

Return mean validation PEHE across folds splits.

Parameters:

  • loader – Data loader providing (X, T, Y) tuples.

  • mu0 – Potential outcomes under control.

  • mu1 – Potential outcomes under treatment.

  • model_config – Model configuration for the ACX model.

  • training_config – Training configuration controlling optimisation.

  • folds – Number of cross-validation folds.

  • device – Device string passed to train_acx().

  • log_dir – Optional directory for TensorBoard logs. Each fold logs to log_dir/fold_i.

  • seed – Random seed for reproducible splits.

Returns:

Mean validation \(\sqrt{\mathrm{PEHE}}\) across folds.

crosslearner.predict_tau_mc_dropout(model: ACX, X: Tensor, *, passes: int = 100) → Tuple[Tensor, Tensor][source]¶

Return mean and standard deviation of MC dropout predictions.

The model is temporarily set to training mode so that dropout layers remain active during the forward passes. Predictions are collected over passes runs and aggregated to obtain an approximate posterior mean and standard deviation for the treatment effect.
