crosslearner.evaluation.evaluate module¶

Evaluation utilities.

crosslearner.evaluation.evaluate.evaluate(model: ACX, X: Tensor, mu0: Tensor, mu1: Tensor) → float¶

Compute PEHE of a model on given data.

Parameters:

  • model – Trained ACX model.

  • X – Covariates (n, p).

  • mu0 – Counterfactual outcome under control.

  • mu1 – Counterfactual outcome under treatment.

Returns:

The square-root PEHE value.

crosslearner.evaluation.metrics.pehe(tau_hat: Tensor, tau_true: Tensor) → float¶

Return sqrt PEHE between predictions and ground truth.

Parameters:

  • tau_hat – Predicted treatment effects.

  • tau_true – True treatment effects.

Returns:

Square-root PEHE.

crosslearner.evaluation.metrics.policy_risk(tau_hat: Tensor, mu0: Tensor, mu1: Tensor) → float¶

Return the policy risk of using tau_hat for treatment decisions.

The function compares the outcome under the policy implied by tau_hat against always choosing the best potential outcome.
