Evaluation utilities.

crosslearner.evaluation.evaluate.evaluate(model: ACX, X: Tensor, mu0: Tensor, mu1: Tensor) → float[source]¶

Compute PEHE of a model on given data.

Parameters:

  • model – Trained ACX model.

  • X – Covariates (n, p).

  • mu0 – Counterfactual outcome under control.

  • mu1 – Counterfactual outcome under treatment.

Returns:

The square-root PEHE value.

crosslearner.evaluation.evaluate.evaluate_dr(model: ACX, X: Tensor, T: Tensor, Y: Tensor, propensity: Tensor) → float[source]¶

Return doubly-robust tau risk for observational datasets.

This estimator compares the model’s CATE predictions against a doubly robust pseudo-outcome constructed from outcome and propensity models. It reduces to PEHE when true counterfactual outcomes are available but can be applied when only observed outcomes are known.

Parameters:

  • model – Trained ACX model.

  • X – Covariates (n, p).

  • T – Treatment indicators (n, 1).

  • Y – Observed outcomes (n, 1).

  • propensity – Propensity scores (n, 1) for treatment.

Returns:

Estimated square-root PEHE using the doubly robust pseudo-outcomes.

crosslearner.evaluation.evaluate.evaluate_ipw(model: ACX, X: Tensor, T: Tensor, Y: Tensor, propensity: Tensor) → float[source]¶

Return IPW tau risk for a dataset without counterfactuals.

The function computes an inverse-propensity weighted pseudo-outcome that is unbiased for the true treatment effect and measures the PEHE between the model predictions and this pseudo-outcome.

Parameters:

  • model – Trained ACX model.

  • X – Covariates (n, p).

  • T – Treatment indicators (n, 1).

  • Y – Observed outcomes (n, 1).

  • propensity – Propensity scores (n, 1) for receiving treatment.

Returns:

Estimated square-root PEHE using IPW pseudo-outcomes.
