crosslearner.datasets.aircraft module¶

Synthetic aircraft dataset based on the Breguet range equation.

crosslearner.datasets.aircraft.get_aircraft_dataloader(batch_size: int = 256, n: int = 8000, seed: int | None = None)¶

Return DataLoader with aircraft performance data.

Covariates are aircraft weight, cruise range, thrust specific fuel consumption, lift-to-drag ratio and cruise speed. Treatment models a modification that increases the lift-to-drag ratio by 10% which in turn reduces fuel burn according to the simplified Breguet range equation.

Parameters:

  • batch_size – Mini-batch size.

  • n – Number of samples.

  • seed – Optional random seed for reproducibility.

Returns:

Tuple (loader, (mu0, mu1)) with true potential outcomes.
